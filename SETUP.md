## Requirements

- Python 3.10+
- A GPU is recommended (CUDA). CPU works but generative answers are disabled.
- Libraries required in the requirements.txt

---

## Option A — Run Locally (MISSING CHAT FUNCTIONALITY IF YOU LACK GPU)

### 1. Install dependencies

```bash
pip install flask transformers torch torchvision sentence-transformers pillow json
```

### 2. Run the server

```bash
python app.py
```

### 3. Open the app

Visit [http://localhost:5000](http://localhost:5000) in your browser.

---

## Option B — Run on Google Colab (GPU, recommended for graders)

### 1. Download `colab/Final_Project_Colab.ipynb`
Upload to colab

### 2. Go to [colab.research.google.com](https://colab.research.google.com)

### 3. Set runtime to GPU
Runtime → Change runtime type → H100 GPU + Max GPU setting → Save

### 4. Generate an API key from ngrok
Sign up and go to Authkeys to find your API key. Input into appopriate locations within `colab/Final_Project_Colab.ipynb`

### 5. Upload all project files
In the left sidebar, click the folder icon and upload:
- `app.py`
- `classifier.py`
- `rag.py`
- `wiki_kb.json` (create `data/` folder first)
- `class_names.json`(input in `data/` folder)
- `models/resnet_plants.pth` (create `models/` folder first)

### 6. Run the cells in order
If you're encountering issues, try running
!pip install --upgrade pandas

## Using the App

1. **Classify** — drag and drop a plant image onto the left panel. The model will identify the species.
2. **Chat** — type a question in the chat box on the right. The system will answer based on the identified species.

### With GPU
Answers are generated by Phi-2, a local language model.

### Without GPU (CPU)
Answers return the Wikipedia entry for the identified species directly from the knowledge base.

---

## Models & External Services

| Component | Model | Source |
|---|---|---|
| Image classifier | ResNet18 fine-tuned on Duke Gardens species | Included (`models/resnet_plants.pth`) |
| Embeddings | `sentence-transformers/all-MiniLM-L6-v2` | Downloaded automatically via HuggingFace |
| Language model | `microsoft/phi-2` | Downloaded automatically via HuggingFace (GPU only) |

Models are cached to `./model_cache` after the first download. No API keys are required.

---

## Notes

- First run will download models (~1.5GB for phi-2, ~90MB for embeddings)
- Subsequent runs load from cache and start in ~1 minute.
- The ngrok URL changes every Colab session. Restart Cell 3 to get a new one.
- Pathnames will likely be changed. For easy install, the data has NOT been provide within the repository, but it is available for download via the `372_Final_Project_CNN.ipynb`
